## 📌 About BOTS

**Boss of the SOC (BOTS)** is a blue-team CTF-style challenge created by Splunk.

It simulates real-world enterprise attack scenarios where analysts must investigate logs using:

- Splunk Enterprise
- Splunk Enterprise Security

Participants analyze datasets, hunt for attacker behavior, and answer forensic-style questions using SPL (Search Processing Language).

## 🧠 Methodology Used

For each challenge, I follow a structured SOC investigation process:

1. Understand the question clearly
2. Identify relevant indexes and sourcetypes
3. Narrow the timeframe
4. Pivot on indicators (IP, username, process, hash, etc.)
5. Build and refine SPL queries
6. Validate findings
7. Document explanation and detection logic
